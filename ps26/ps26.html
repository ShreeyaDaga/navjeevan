PICT B.E. Course Structure

First Year (FE)

  • Mathematics: Fundamental engineering math concepts.
  • Physics: Engineering physics principles.
  • Chemistry: Engineering chemistry basics.
  • Engineering Mechanics: Statics and dynamics introduction.
  • Computer Fundamentals: Basics of computer systems.

Second Year (SE)

  • Computer Programming: C/C++ programming techniques.
  • Data Structures: Stack, queue, tree, graph, etc.
  • Digital Electronics: Logic gates and circuits.
  • Discrete Mathematics: Sets, relations, graphs.
  • Microprocessor: Intel 8085/8086 architecture.

Third Year (TE)

  • Algorithms: Sorting, searching, dynamic programming.
  • Operating Systems: Processes, threads, scheduling.
  • Computer Networks: OSI/TCP-IP model, protocols.
  • Software Engineering: SDLC, agile, UML.
  • DBMS: SQL, normalization, transactions.

Final Year (BE)

  • Software Engineering: Advanced SE practices.
  • Artificial Intelligence: Search, knowledge representation.
  • Cloud Computing: Cloud models, virtualization.
  • Machine Learning: Supervised, unsupervised learning.
  • Data Science: Data analysis, visualization.
